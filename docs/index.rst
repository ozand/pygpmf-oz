Welcome to pygpmf_oz's documentation!
=====================================

**pygpmf_oz** is a Python library for extracting and processing GoPro Metadata Format (GPMF) data from GoPro video files.

Features
--------

* Extract GPS data (latitude, longitude, altitude, speed)
* Export GPS tracks to GPX format
* Visualize GPS tracks on maps with OpenStreetMap
* Parse GPMF Key-Length-Value structures
* Support for Python 3.9-3.13
* Cross-platform (Windows, macOS, Linux)
